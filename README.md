# Movie Stats

A personal movie statistics dashboard built from an IMDB watchlist, enriched with TMDB data.

## What it does

Visualizes ~1674 watched films across multiple views: genres, directors, actors, writers, composers, cinematographers, countries, and timeline. Each view has drill-down detail pages that cross-link to related people and films. Hosted on GitHub Pages, mobile-first, IMDB-inspired dark theme.

## Running the enrichment script

The enrichment script fetches movie data from [TMDB API](https://www.themoviedb.org/) and produces `data.json` which the web app reads.

### 1. Get a TMDB API key

Create a free account at [themoviedb.org](https://www.themoviedb.org/), go to **Settings → API**, and request an API key. You'll get an **API Read Access Token** (the long JWT string).

### 2. Create a `.env` file

In the project root, create a file called `.env` (it is gitignored and will never be committed):

```
TMDB_BEARER_TOKEN=your_read_access_token_here
```

### 3. Set up Python environment

```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install requests python-dotenv
```

### 4. Run the test script first (10 movies)

```bash
python scripts/test_enrich.py
```

### 5. Run the full enrichment

```bash
python scripts/enrich.py
```

This fetches all ~1674 movies from TMDB (roughly 2–3 minutes). Outputs `data.json`.

## Hosting on GitHub Pages

Push the repo to GitHub. In the repo settings, enable GitHub Pages from the `main` branch root. The app runs entirely from static files — no server needed.
